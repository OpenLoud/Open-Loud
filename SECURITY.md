# Security Policy

## Reporting a Vulnerability

Please do NOT open a public GitHub issue for security vulnerabilities.

Email: security@loud.gg

We will respond within 48 hours and coordinate a fix before public disclosure.

## Scope

- Circom circuits in `packages/circuits/`
- Solidity contracts in `packages/contracts/`
- TypeScript SDK in `packages/sdk/`

Phase 6 (external audit) is planned before mainnet. See [docs/workflow.md](docs/workflow.md).
