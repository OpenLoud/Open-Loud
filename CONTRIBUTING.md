# Contributing to loud

We welcome contributions! loud is open-source and community-driven.

## Getting Started

```bash
git clone https://github.com/yourusername/loud
cd loud
pnpm install
pnpm setup
pnpm dev
```

## Areas We Need Help

- Circom circuit optimization
- Solidity gas optimization
- TypeScript SDK improvements
- Documentation
- Frontend (Next.js)

## Pull Request Process

1. Fork the repo
2. Create a feature branch
3. Write tests for your changes
4. Open a PR with a clear description

## Code of Conduct

Be respectful. We are building open privacy infrastructure for everyone.
